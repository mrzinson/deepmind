# DeepMind AI Platform

DeepMind is a sophisticated Somali AI platform offering specialized chat and voice interactions for Business, Romance, and Education.

## Features
- **DeepMind Hybrid Models:** Powered by the latest AI technologies.
- **Specialized Chat Hub:** Business Expert, Education Pro, and Romance Guru agents.
- **Voice Interactions:** Real-time Somali voice calls with Cali and Layla.
- **One-Click Deploy:** Optimized for Render with Blueprint support.

## Deployment to Render
1. Connect this repository to your Render account.
2. Render will automatically detect the `render.yaml` file.
3. Click "Create Web Service".
4. The API keys are pre-configured in the Blueprint, but you can override them in the Render Dashboard.

## Local Development
1. Install dependencies:
   ```bash
   npm run install-all
   ```
2. Run development server:
   ```bash
   npm run dev
   ```
